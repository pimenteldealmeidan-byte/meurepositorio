# Meu projeto da Semana 02 de BackEnd

1.	Qual é a importância de usar repositórios remotos, como GitHub ou GitLab, em projetos colaborativos?
R= O uso de repositórios remotos como GitHub ou GitLab é o que transforma o trabalho individual em um esforço de equipe organizado e seguro.


2.	Como o comando git pull facilita a colaboração em um projeto?
R= Ele facilita a colaboração ao garantir que você não trabalhe sobre uma versão obsoleta do código.

3.	Por que é importante realizar frequentemente o push de alterações para um repositório remoto?
R= Realizar o push com frequência é como apertar o "salvar" em um jogo: você garante que seu progresso não seja perdido e que seus aliados saibam o que está acontecendo.
